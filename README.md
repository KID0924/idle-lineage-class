# ☁️ 放置天堂 - 雲端存檔同步與快取引擎 (`klh_database.js`)

本模組為《放置天堂 - 以血為盟》同人遊戲的整合式雲端備份引擎。支援多種雲端儲存方式，並搭載主動式本地快取清理與安全防護機制，讓您的遊戲進度安全無虞、隨時隨地輕鬆同步。

---

## 🚀 核心功能介紹

### 1. 📂 三大儲存模式
*   **本地模式 (單機)**
    - 將存檔直接保存在本機瀏覽器的 `localStorage` 中，適合不需要跨裝置遊玩的單機玩家。
*   **JSONBlob 雲端備份模式**
    - ➔ **特色**：免費、免註冊，點擊即可快速建立專屬網址。
    - ➔ **技術**：傳輸時自動採用 **LZ-String 演算法**進行 Base64 二進位高壓縮，節省流量並防止代碼在雲端明文顯示。
    - ➔ **適合人群**：適合**存檔較小（如初期玩家、進度較少）**的使用者。
*   **Firebase 雲端同步模式**
    - ➔ **特色**：使用 Google 的 Firebase Realtime Database 建立個人專屬的即時資料庫，高穩定且安全。
    - ➔ **功能**：
      * **一鍵建立存檔**：在面板直接點選「建立存檔」即可在 Firebase 上初始化空白進度節點。
      * **本機存檔還原**：更換登入其他存檔名稱（Sync ID）後，面板會即刻出現黃星還原按鈕，可一鍵快速切回您的主存檔。
      * **免頻寬浪費設計**：重整網頁時不會主動拉取 Firebase 數據以防浪費免費流量，僅在手動登入或遊戲內存檔時與雲端通訊。
    - ➔ **適合人群**：適合**存檔較大（遊玩進度極深、遊戲數據龐大）**的使用者。

### 2. 🧹 智慧快取清理器 (Cache Sweeper)
*   **自動資源回收**：為避免玩家切換帳號或金鑰時在瀏覽器中累積大量垃圾檔案，本系統會在**切換模式**、**變更金鑰**或**重新整理網頁**時，自動在背景清理所有與當前登入金鑰無關的舊快取。
*   **終極空間控制**：不論您曾經登入過多少個不同的雲端帳號，瀏覽器內始終只會保留「本地單機存檔」與「當前登入的雲端存檔」**最多 2 套**，完全不留多餘垃圾。

### 3. 🔍 本地快取即時監視器 (Storage Inspector)
*   **即時可視化**：在「雲端存檔設定」標題旁點擊 `🔍 快取`，即可打開超輕量浮動視窗。
*   **資訊一目了然**：
    - 🟢 **綠色** 標示本機的單機存檔與倉庫。
    - 🟠 **橘色** 標示當前雲端各槽位、倉庫與空白旗標的體積（以 `KB` 顯示）。
*   **強制手動清理**：監視器底部提供紅色「強制手動清理」按鈕，可一鍵強制淨化快取空間。

### 4. 🔒 安全防覆寫機制
*   **防洗白保護鎖**：若連線中斷或雲端讀取失敗，系統會自動阻斷隨後的「自動存檔」動作，並鎖定遊戲畫面提示，防止本地的新手空白進度上傳並意外覆蓋掉雲端舊存檔。

## 🛠️ 使用與設定方法

### 1. 啟用 JSONBlob 雲端備份
1. 開啟遊戲，點擊 **☁️ 雲端存檔設定** 按鈕。
2. 切換到 **Jsonblob** 標籤頁。
3. 按照底部說明，前往 [JSONBlob 官網](https://jsonblob.com) 註冊並點擊 **Save** 產生一組金鑰。
4. 將該網址或金鑰（格式如 `019f3dad-...`）貼入輸入框中。
5. 點擊 **登入** 即可開始同步。

### 2. 啟用 Firebase 雲端同步
1. 前往 [Firebase 官網](https://firebase.google.com) 點擊 **Go to console** 並登入。
2. 點擊 **新增專案**，隨意填寫專案名稱（可關閉 Google Analytics 以免拉長建立時間）。
3. 專案建立後，在左側選單點選 **Realtime Database**，並點擊 **建立資料庫**。
4. 位置保留預設（美國），安全模式選擇鎖定或測試模式皆可。
5. 資料庫建立完成後，複製首頁顯示的資料庫網址（格式如：`https://[專案名稱].firebaseio.com`）。
6. 前往資料庫的 **規則 (Rules)** 頁籤，將內容修改為：
   ```json
   {
     "rules": {
       ".read": true,
       ".write": true
     }
   }
   ```
   修改完成後點擊 **發布**。
7. 回到遊戲中的 **☁️ 雲端存檔設定**，切換至 **Firebase** 標籤頁。
8. 貼入剛剛複製的 **伺服器網址**，並在下方自訂一個您的 **存檔名稱 (Sync ID)**（如 `my_save_slot`），或者點擊 **建立存檔** 建立全新的存檔名稱。
9. 點擊 **登入**，即可完美享受無限制的即時雲端同步！
